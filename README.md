# 文字密室 🔐

基于 HarmonyOS ArkTS 开发的文字冒险解谜游戏。纯文字交互，在封闭空间中寻找线索、收集道具、解开谜题逃出去。

## 作者

**薛洋**

## 游戏特色

- 🧠 **纯文字解谜**：没有花哨的UI，只有文字和点击，回归解谜本质
- 🗺️ **正交地图**：节点按上下左右正交展开，配合连线，结构清晰
- 🎒 **道具系统**：收集道具 → 装备 → 对目标使用，经典冒险玩法
- 📓 **线索记事本**：自动记录关键线索，随时查看
- 💡 **动态提示**：卡关时自动给出下一步提示
- 🏆 **成就系统**：通关解锁成就，进度持久化保存
- 📱 **手势操控**：支持拖拽平移和双指缩放地图

### 第一关：深夜加班

> 每天都涉及无休止的加班，今天困得不行了，趴着睡了一会，醒来天已经完全黑了……

**玩法：**
- 探索办公室 → 工位找到邮件线索 → 获取会议室密码
- 会议室找到应急手电 → 照亮杂物间
- 杂物间找到备用钥匙 → 打开大门逃脱

### 第二关：回魂夜

> 午夜十二点，大楼停电了，你必须要从一楼的走廊逃出去。

**玩法：**
- 探索走廊 → 工具箱找到撬棍 → 暴力破门进保安室
- 保安桌找到值班日记（线索）和电梯卡
- 尽头的镜子发现红衣女人（纯叙事）
- 电梯刷卡逃生

### 第三关：精神病院

> 你在一张沾满血迹的病床上醒来，这里的人都不正常，你必须离开。

**玩法：**
- 病床找到血字纸片 → 线索：03颗红药，04颗蓝药
- 护士站药柜输入密码 0304 → 获得手术刀
- 治疗室用手术刀切开束缚衣 → 获得A区门禁卡
- 铁栅栏门刷卡逃生

### 第四关：特大案件

> 你作为特别调查员，来到了连环杀人案的最后现场，找出决定性的证据！

**玩法：**
- 受害者遗体找到收据（尾号77） + 书桌日历（红圈24号） → 密码 7724
- 密码抽屉获得档案室钥匙 → 开门进入档案室
- 紫光灯照射保险箱 → 发现按键 1、8、9（密码1998）
- 获得加密U盘 → 警用终端上传结案

## 技术栈

- **框架：** HarmonyOS ArkTS (API 12)
- **构建：** hvigor
- **状态管理：** @State + @StorageLink
- **动画：** animateTo + transition
- **手势：** PanGesture + PinchGesture

## 核心架构

```
GameNodeConfig (数据层)
├── id: 节点唯一标识
├── label: 显示名称
├── x, y: 坐标位置
├── parentId: 父节点（用于连线）
├── bgColor: 自定义背景色
├── isVisible: 条件可见性
└── onClick: 点击逻辑

GameContext (引擎层)
├── getClick/setClick: 点击计数器
├── getFlag/setFlag: 布尔状态标记
├── setPrompt: 提示消息
├── recordClue: 记录线索
├── useItem/consumeItem/grantItem: 道具系统
├── focusCamera: 镜头聚焦
├── openPasswordUI: 密码输入
└── triggerAchievement: 成就触发
```

## 设计思路

### 坐标布局

采用正交收敛布局，以起点为中心向四个方向展开：
- **向上**：探索/调查分支
- **向左**：辅助/工具分支
- **向右**：核心道具分支
- **向下**：逃生/通关分支

所有子节点与父节点间距统一为 160px，避免坐标发散。

### 道具流转

每关设计两条独立的道具链，最终汇聚到出口：
- 第三关：病床线索 → 药柜 → 手术刀 **+** 治疗室 → 束缚衣 → 门禁卡 → 出口
- 第四关：遗体收据 + 书桌日历 → 密码抽屉 → 钥匙 **+** 档案室 → 紫光灯 → 保险箱 → U盘 → 终端

### 反 AI 检测

所有文案都经过人工润色，避免 AI 味：
- 口语化表达
- 具体细节（数字、颜色、气味）
- 情绪化描述
- 不规则句式长度
